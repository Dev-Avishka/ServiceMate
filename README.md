# Sevice Mate

An app that ca remind you to service your car.


## dependecies 
Sqflite

Couportino Icons

## Getting Started
```bash
git clone https://github.com/Dev-Avishka/ServiceMate.git
```
then run 

```bash
flutter doctor
```
then
```bash
flutter pub add
```
then setup the project and run

## Or if you want to download the app

### android
```bash
flutter build apk
```
### Ios
```bash

flutter build apk --split-per-abi

```
